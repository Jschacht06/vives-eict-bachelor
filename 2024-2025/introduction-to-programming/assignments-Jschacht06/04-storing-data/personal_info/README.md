# Personal information

Start by creating a new Console Application.

Now make the following variables:
* a `string` variable to hold your name.
* a `string` variable to hold your firstname.
* a `int` variable to hold your age.
* a `string` variable to hold your gender.
* a `int` variable with `2023`

Now output the information to the console in the following format:

```text
This is my personal information

Name        <your name>
Firstname   <your firstname>
Age         <your age>
Birth year  <year you are born>
Gender      <the gender associated with you>
```

## Birth year

Start with just calculating your birth year from `2023 - age`.

If that works and you are up for a challenge try to adjust your program so it automaticly knows what year we are now.
