# Biggest Number in a List

Imagine you had a list of numbers: `1, 18, 5, 12, 11, 0, 33, 105, 20, 88, 9, 32`. How would you determine the biggest number? Document the algorithm in plain English how you would achieve this.


I would take the first number from the list and then compare it to another number from the list and if that number is bigger then take that number  and delete the previous number from the list then repeat this process until you have the biggest number
