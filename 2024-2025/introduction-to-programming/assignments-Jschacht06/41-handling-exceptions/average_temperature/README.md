# Average Temperature

We will be using command line arguments in this console application.
You can add these by going to Project > AvgTemp Properties, in the new window go to Debug > General en click on `Open debug launch profiles UI`, again a new window pops up and in the first line you can enter the command line arguments.

First leave it blank an see what happens.

Next fill in some numbers, for example `10 20 9 17 4 16` an you will get the average.

Next change one number to a letter and see what happens.

Adjust the code so errors like deviding by zero and non numeric arguments are dealt with and proper feedback is given to the user.
